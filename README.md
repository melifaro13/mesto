# Проект: Место

## Описание проекта:
Страница сайта с возможностью менять информацию профиля, загружать фотографии и ставить лайки.

## Используемые приемы для создания проекта:
+ *HTML/CSS*
+ *Файловая структура БЭМ - Nested*
+ *Адаптивная верстка*
+ *Grid Layout*
+ *Flex*
+ *Абсолютное позиционирование*
+ *Медиазапросы*
+ *Оптимизация шрифтов*
+ *Создание popup окна*
+ *Основы JavaScript*
